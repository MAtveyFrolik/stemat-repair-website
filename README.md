# Сайт компании «СтеМат»

**Современный landing page для компании по срочному ремонту промышленного оборудования**

Сайт создан для привлечения клиентов, занимающихся обслуживанием лазерных станков, станков с ЧПУ, гибочного оборудования, компрессоров и лазерной сварки.

## Особенности

- Адаптивный дизайн (отлично выглядит на мобильных и десктопе)
- Фиксированная «стеклянная» навигационная панель с эффектом blur
- Web Components (`CustomNavbar` и `CustomFooter`)
- Плавная прокрутка к секциям
- Анимированные карточки услуг
- Удобное бургер-меню для мобильных устройств
- Призывы к действию (CTA) для подачи заявок

## Технологии

- **HTML5**
- **Tailwind CSS**
- **JavaScript (ES6+)**
- **Web Components** (Custom Elements + Shadow DOM)
- Feather Icons

## Демо

[Открыть сайт](https://твоё-имя.github.io/stemat-repair-website/)  
(ссылка появится после включения GitHub Pages)

## Как запустить локально

1. Склонируй репозиторий
2. Открой файл `index.html` в браузере
